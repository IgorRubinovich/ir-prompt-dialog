# ir-native-input-reflector

**Polymer 1.0 control to to simplify making of pop-up input dialogs.**

## How to use it
	<ir-prompt-dialog
		target="id1,id2,..."
		open-event-source="s1,s2,..."
		open-event="e1,e2,..."
		value-attr="va1,va2,..."
		close-event="ev1,ev2,.."
		
		content-value-source="id"
		ok-event="ev1"
		cancel-event="ev2"
	>
	</ir-prompt-dialog>
